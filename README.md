## What is Unscripted Souls?

Unscripted Souls is an artificial life simulation (ALife) developed with Godot that explores a simple yet ambitious question:

**What happens when digital beings stop following predefined scripts and begin to live their own lives?**

Instead of traditional NPCs that repeat predefined behaviors, Unscripted Souls seeks to create autonomous beings capable of:

- Remembering past experiences
- Forming relationships
- Making independent decisions
- Developing personalities
- Pursuing personal goals
- Adapting to changing environments
- Participating in emerging societies

The project combines simulation systems, local AI models, memory architectures, and agent-based design to create a world that evolves even without player interaction.

-

## Vision

Most game worlds revolve around the player.

Unscripted Souls takes a different approach:

> A world where every citizen has their own story, even when no one is watching.

Characters wake up, work, socialize, learn, remember, cooperate, argue, make friends, and shape their environment through countless interactions.

The goal is not to create smarter NPCs.

The goal is to create digital life.

-

## Main Features

### 🧠 Permanent Memory

Agents remember:

- Conversations
- Important events
- Relationships
- Personal experiences

Memories influence future decisions and behaviors.

-

### ❤️ Dynamic Relationships

Citizens develop:

- Friendships
- Rivalries
- Trust
- Respect
- Affection
- Fear

Relationships develop naturally through interactions.

---

### 🎭 Personality-Based Behavior

Each agent possesses unique characteristics that influence:

- Decision-making
- Communication style
- Social preferences
- Emotional responses

---

### 🌱 The Evolving Society

The world is designed to support:

- Social groups
- Communities
- Leadership structures
- Cooperation
- Conflict
- Cultural evolution

---

### 🤖 Local AI Integration

The project uses local AI models to improve:

- Dialogue generation
- Decision-making support
- Context understanding
- Long-term memory processing

It does not require cloud dependency.

--

### 🏠 Real-World Connectivity

Experimental integrations with Home Assistant allow digital entities to interact with real-world devices and events.

This opens up fascinating possibilities for interaction between physical and virtual environments.
